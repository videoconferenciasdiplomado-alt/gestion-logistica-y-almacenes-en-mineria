14 Métodos y herramientas para pronósticos
▼
14.1 Pronósticos cuantitativos y cualitativos
▼
Enfoques cuantitativos: series de tiempo, regresión (simple o exponencial), modelos econométricos; se recomiendan métodos sencillos (regresión simple, medias móviles) para uso operativo.
Uso cuando hay datos históricos, mercado estable y productos estandarizados; aplicable a expansiones y compras/importación con economista.
Enfoques cualitativos: juicio de expertos, Delphi, estudios de mercado y análisis de escenarios.
Útiles cuando faltan datos históricos, hay productos nuevos o mercados volátiles; subjetivos y basados en experiencia.
Recomendación práctica: empezar con métodos simples y emplear modelos más complejos cuando exista historial suficiente y necesidad de precisión.
Ver en YouTube (01:44:50)
14.2 Técnicas y herramientas para pronóstico
▼
Técnicas: medias móviles (for datos limitados), regresión (causas variables), proyección por historial de mantenimiento (predicción de fallas según horas de operación).
Ejemplo operativo: usar historial de mantenciones para estimar fallo a 5.000 horas y programar pedidos anticipados en mes 4 para evitar paros.
Cualitativos y software: simulaciones, Big Data, recopilación tercerizada; Delphi para consenso de expertos; análisis de escenarios para mercados inciertos.

15 Pronóstico de series temporales mediante medias móviles
▼
15.1 Promedio móvil simple y aplicación práctica
▼
El procedimiento implica elegir un valor n como tres o seis para calcular el promedio de los periodos anteriores y desplazar la ventana móvil mes a mes usando datos históricos para proyectar el futuro.
Las medidas de desempeño como el error medio y el error medio absoluto permiten evaluar el ajuste comparando diferentes valores de n para encontrar el equilibrio entre suavizado y reacción.
El uso operacional sugiere iniciar con métodos simples ajustando según la disponibilidad de datos y graficando las series reales frente a las pronosticadas para detectar visualmente cualquier desajuste.
Ver en YouTube (02:00:04)
15.2 Herramientas de software y extensiones
▼
La utilidad principal radica en detectar series y rellenar valores faltantes para generar pronósticos automáticos siempre que los datos cumplan con el formato de fecha y numérico requerido.
La configuración necesita conexión a un servidor de minería de datos para crear una base temporal y generar modelos que resultan en predicciones integradas en tablas de Excel.
Las limitaciones más comunes incluyen errores por la configuración del servidor o permisos de acceso que a menudo requieren ajustes técnicos en SQL para habilitar la minería de datos.
